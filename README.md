# 名刺エディター（ミッシェル）

ブラウザだけで動作する名刺プレビュー・編集ツールです。`index.html` をブラウザで開くだけで使えます。サーバー不要、インストール不要。

## 使い方

1. `index.html` をブラウザで開く
   - またはホストしている [michele](https://michele.pages.dev/) にて使用
2. 左側のパネルで各項目を編集すると、右側の名刺プレビューにリアルタイム反映される
3. 印刷時はパネルが非表示になり、名刺サイズ（91mm × 55mm）で出力される

## 機能

### 編集項目

| セクション     | 内容                         |
| -------------- | ---------------------------- |
| 氏名           | 日本語氏名・英語氏名・役職   |
| 事業・サービス | カンマ区切りで複数タグを指定 |
| 会社           | 会社名                       |
| 連絡先         | メール・電話・Web・住所      |

- 各項目左のトグルスイッチでカード上の表示/非表示を切り替えられる
- 役職・事業タグはカンマ区切りで複数指定可能（例: `Senior Engineer, Tech Lead`）
- アバターアイコンをクリックして写真をアップロードできる

### データの保存・読み込み

パネル下部の「データ」セクションからJSON形式でデータを保存・読み込みできます。

- **JSONで保存** — 現在の入力内容・トグル状態・アバター画像を `meishi.json` としてダウンロード
  - アバター画像もBASE64で保存されます。
- **JSONを読み込む** — 保存した `.json` ファイルを選択して状態を復元

### 印刷

ブラウザの印刷機能（`Ctrl+P`）を使用します。印刷時はパネルが自動的に非表示になり、名刺サイズで出力されます。

## 技術構成

- 純粋な HTML / CSS / JavaScript（フレームワーク・ビルドツール不要）
